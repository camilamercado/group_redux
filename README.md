# Shopify Redux
Branched out of dan_version

Redux toolkit for implementing Shopify Javascript Buy SDK


# Setup

Install the npm dependencies and run the webpack dev server:

```bash
npm i
node_modules/.bin/webpack-dev-server
open localhost:8080
```
